动态组件的强制数据渲染

dynamic component vue force re-render

问题：切换组件时re-render，变数据时却不

解决：表明唯一身份，加个 :key 参数

<component ... :key="{MyMethod()}"></component>
MyMethod(){
    return +new Date
}

PS：$forceUpdate是没用的，因为根本没有更新props
