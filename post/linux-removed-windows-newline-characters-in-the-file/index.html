目录

  1. 1VI
  2. 2SED方法

VI

对某个文件进行处理

vi filename
:%s/^M/\r/g

SED方法

PERL方法

可对多文件批量处理

perl -p -i -e "s/^M//g" `find .`

find . 可以替换为任何文件或者多个文件，它的作用为提取所需的文件名。

^M字符需要使用Ctrl+v+m来输入
